# 🎱 3D Snooker Game

A browser-based 3D snooker game built with Three.js.

## How to Play

1. **Aim**: Click and drag on the screen to aim
2. **Power**: The further you drag, the more power
3. **Shoot**: Release to hit the cue ball
4. **Score**: Pot red balls (1 point) first, then colored balls!

## Controls

- 🖱️ **Drag** to aim
- 🎯 **Release** to shoot
- 👁️ **Toggle View** button for top-down view
- 🔄 **New Game** to restart

## Deployment

This game can be deployed to GitHub Pages:

1. Create a new repository on GitHub
2. Upload this `index.html` file
3. Go to Settings → Pages
4. Select "main" branch and save

## Technologies

- Three.js for 3D rendering
- Vanilla JavaScript
- HTML5/CSS3
